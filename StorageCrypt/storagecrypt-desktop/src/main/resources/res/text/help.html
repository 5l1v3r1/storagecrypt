Keystore creation

The first time you launch the app, a secret key is generated and stored in a secure keystore.

You must provide a password, to protect this keystore. Remember it, you will need to type it every time you launch the app.

Link your cloud accounts

At first, you only have one storage location where you can encrypt files. It is shown as "Unsynchronized files". As you can guess, the files you encrypt there stay on your device.

If you want your encrypted files to be uploaded to Dropbox, for example, you have to link your Dropbox account first.

To do that, use the button ("Link a new cloud account") in the tool bar at the top of the screen.

You can choose between "Google Drive", "Dropbox", "Box", "HubiC" or "One Drive".

You also have to choose which key you want to use by default for this account (you can generate as many keys as you like, but you can also only use the default key).

Then you will have to provide your user name and password for your cloud account (not the password of the keystore).

After linking an account, StorageCrypt will search for any already encrypted files, and download them if any.

Then, you will see a new folder next to the "Unsynchronized files", which represents your linked account.

Encrypt files

To encrypt a file, you first need to go to the location where you want the encrypted file to be placed, for example "Unsynchronized files".

Then, use the button ("Encrypt") in the tool bar at the top of the screen.

Files can also be dragged from your file manager.

After file selection, you will have to select which key will be used to encrypt the selected file(s). You can have as many different keys as you like, or use only the default one.

If you encrypted the file inside the "Unsynchronized files" folder, that's it.

If you encrypted it into a folder related to a cloud account, the file will be uploaded to this account.

Files download and upload

Downloading and uploading files may imply a lot of bandwidth consumption, so you can choose if you want to wait for wifi connection to download, or upload files.

This is the default configuration of the app, so when encrypted, the files will be marked for upload later when you connect to a wifi network.

Note that the app needs to be running to perform uploads or downloads, even if it runs in the background.

If you exit the application, all running transfers will be stopped, and planned again on next launch.

Create folders

In order to manage your encrypted files more easily, you can create folders on your encrypted storage.

To do that, use the button ("Create en empty folder") in the tool bar at the top of the screen.

Give the name of the new folder, and select the key which will be used by default to encrypt files in this folder.

Browse documents

You can navigate through folders by clicking them. You can also trigger various actions on folders with a right click.

On the top of the screen, you will see the name of the current folder. Click it to go back to the parent folder.

You can also right click on this current folder name to show the other folder actions.

Click on encrypted files to open them (they will be automatically decrypted on demand). Also, right clicking on files will show more options.

Manage your keys

If you select the "Manage secret" menu, you will see a list of the existing keys.

Right click on one of them to rename or delete it. Renaming a key doesn't break the files you already encrypted with key.

Warning : if you delete a key, though, you will NOT be able to recover the files encrypted with this key.

In the menu, you will see options to export your keys, for example if you want to access the encrypted files on another device.

The keys you choose to export are saved in a password protected keystore.

Transfer this keystore to your other device, and use the "Import keys" option to import the keys you want.

In the menu, you also have options to create a new key, which you can use to encrypt some files.

The "Change keystore password" menu makes it possible to change the app keystore password. You will have to provide the old password, and the new password.

Settings

If you select the "Settings" menu, you can modify the app settings.

Select the connection mode you wish to use :

  • No proxy
  • Use system proxies
  • Manual configuration

If you choose the last option, you will also need to specify your proxy address and port.

Check remote files

At the top of the main screen, at the right of the current folder name, you will see 2 buttons.

Clicking the first one will check if there are new files uploaded on all your remote accounts.

If any, they will be created locally, and a download will be planned. The color of this button will reflect the state of this operation.

The second button reflects the current file synchronisation state (deletion, upload and download).

Clicking these buttons will either launch the synchronization action if none is taking place (black indicator), or display the progress window of the running action (any other color).

Note that icons reflecting these states will also appear near each file in the list.

On the progress windows, you usually have two buttons, to cancel the action, or pause it, then resume it later.
