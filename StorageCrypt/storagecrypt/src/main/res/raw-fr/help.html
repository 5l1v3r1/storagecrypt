Création du magasin de clés

Lors du premier lancement de l'application, une clé de chiffement est générée et stockée dans un magasin de clés sécurisé.

Vous devez fournir un mot de passe, pour protéger ce magasin de clés. Ne l'oubliez pas, vous devrez le saisir à chaque fois que vous lancez l'application.

Si vous trouvez ce comportement fastidieux, vous pouvez demander au système de garder l'application active en arrière plan.

Ainsi vous n'aurez pas à saisir le mot de passe, à moins d'utiliser l'option "Quitter" dans le menu, ou de redémarrer votre appareil.

Ce paramètre "Garder l'application active" se trouve dans le menu "Paramètres".

Associer vos comptes de stockage dans le cloud

Au début, vous n'avez qu'un seul emplacement pour chiffrer vos dossiers. Il est nommé "Fichiers non synchronisés". Comme vous pouvez le deviner, les fichiers chiffrés ici restent sur votre appareil.

Si vous souhaitez que vos fichiers chiffrés soient synchronisés avec Dropbox, par exemple, vous devez d'abord associer votre compte dropbox.

Pour cela, utilisez le bouton ("Associer un compte de stockage") en haut de l'écran ou dans le menu.

Vous pouvez choisir comme type de compte "Google Drive", "Dropbox", "Box", "HubiC" ou "One Drive".

Vous devez aussi sélectionner quelle clé de chiffrement sera utilisée par défaut pour ce compte (vous pouvez gérérer plusieurs clés ou en utiliser une seule, selon vos préférences).

Il vous est ensuite demandé de saisir les informations de connexion à votre compte.

Une fois votre compte associé à l'application, si des fichiers chiffrés s'y trouvent déjà, ils seront repérés et téléchargés.

Vous remarquerez alors un nouvel emplacement de stockage, aux côtés des "Fichiers Locaux", qui représente les fichiers synchronisés avec votre compte.

Chiffrer des fichiers

Pour chiffrer un fichier, vous devez auparavant vous placer à l'endroit où vous voulez le placer, par exemple "Fichiers non synchronisés".

Ensuite, utilisez le bouton ("Chiffrer") en haut de l'écran ou dans le menu.

Vous avez le choix de chiffrer un seul ou plusieurs fichiers. Pour chiffrer un unique fichier, la sélection se fera avec l'outil de sélection de fichiers natif d'android.

Si vous choisissez de chiffrer plusieurs fichiers à la fois, vous devrez utiliser un dialogue de sélection spécifique, car celui du système ne permet pas la sélection multiple.

Après la phase de sélection, vous devez choisir la clé avec laquelle le fichier sera chiffré. Vous pouvez utilisez différentes clés ou vous contenter de la clé par défaut pour tous vos fichiers.

Si la destination se trouve à l'intérieur du dossier "Fichiers Locaux", il ne se passera rien de plus.

Si elle se trouve à l'intérieur d'un emplacement associé à un compte de stockage dans le cloud, les fichiers qui y sont chiffrés seront synchronisés sur ce compte.

Download et upload des fichiers

Le transfert de fichiers peut impliquer une grande consommation de bande passante, il est donc possible d'attendre une connexion à un réseau wifi pour transférer les fichiers.

C'est la configuration par défaut de l'application, ainsi, quand un fichier est chiffré, il est marqué pour être transféré ultérieurement, lorsque vous serez connecté à un réseau wifi.

Notez que l'application doit être lancée (même en arrière plan) pour que les transferts se fassent.

Si voud utilisez le bouton "Quitter" tous les transferts en cours seront abandonnés, et planifiés à nouveau pour le prochain lancement.

Créer des dossiers

Afin de gérer plus facilement vos fichiers chiffrés, vous pouvez créer des dossiers sur vos emplacements de stockage.

Pour cela, utilisez le bouton ("Créer un dossier vide") en haut de l'écran ou dans le menu.

Saisissez le nom du nouveau dossier, et sélectionnez la clé qui sera utilisée par défaut pour y chiffrer des fichiers.

Consulter les documents chiffrés

Vous pouvez naviguer dans les différents dossiers en appuyant brièvement dessus. D'autres actions sont possibles si vous laissez votre doigt dessus plus longtemps.

En haut de l'écran, est affiché le nom du dossier actuel. Appuyez dessus pour revenir au dossier parent.

Vous pouvez aussi maintenir la pression plus longtemps pour afficher les actions supplémentaires.

Appuyez brièvement sur les fichiers chiffrés pour les ouvrir (ils seront déchiffrés automatiquement). Appuyer longuement sur les fichiers vous permettre d'exécuter d'autres actions.

Gérer vos clés

Si vous sélectionnez le menu "Gérer les clés", la liste des clés sera affichée.

En maintenant votre doigt sur l'une d'elle, vous pourrez la renommer ou la supprimer. Renommer une clé ne vous empèchera pas re relire les fichiers que vous avez chiffrés avec.

Attention : si vous supprimez une clé, par contre, vous ne pourrez PAS récupérer les fichiers que vous avez chiffré avec.

Dans le menu, vous trouverez les options permettant d'exporter des clés, par exemple si vous voulez accéder à vos fichiers chiffrés sur un autre appareil.

Les clés que vous choisissez d'exporter seront sauvées dans un magasin de clés protégé par mot de passe.

Transférez ce magasin de clés sur votre autre appareil, et utilisez l'option "Importer des clés" pour importer les clés que vous souhaitez.

Dans ce menu, vous trouverez également une option pour générer une nouvelle clé, que vous pourrez utiliser pour chiffrer certains fichiers.

L'option "Changer le mot de passe" vous permet de changer le mot de passe du magasin de clés. Vous devrez saisir l'ancien mot de passe et le nouveau.

Paramètres de l'application

Si vous sélectionnez le menu "Paramètres", vous pourrez modifier les paramètres de l'application.

Sélectionnez l'option "Garder l'application active" si vous voulez garder l'application active en arrière plan.

Ainsi, les transferts de fichiers continueront si vous utilisez une autre application, et vous n'aurez pas à saisir le mot de passe à chaque lancement.

Dans la section "Synchronisation", vous pouvez choisir quelles actions peuvent être effectuées seulement lorsque vous êtes connectés sur un réseau wifi, ou sur tout type de réseau.

La création et la suppression de fichiers distants consomment peu de bande passante, alors que le download et l'upload peuvent être très consommateurs, choisissez donc sagement.

La configuration par défaut permet de créer ou de supprimer les fichiers sur tout type de réseau, alors que le download et l'upload sont limités au x réseaux wifi.

Les derniers paramètres demandent un peu plus d'explications.

Lorsque vous ouvrez un fichier chiffré, StorageCrypt peut passer les données déchiffrées directement aux autres applications, ou les déchiffrer d'abord dans un dossier temporaire, puis leur passer le fichier déchiffré.

La première possibilité est plus sécurisée, mais toutes les applications ne l'acceptent pas.

La seconde option implique une durée de lancement plus longue, car les fichiers doivent être déchiffrés intégralement avant de les ouvrir.

Elle peut également être considérée comme moins sûre, car le fichier déchiffré est stocké en clair sur l'appareil pendant sa lecture.

Le fichier déchiffré sera supprimé dés qu'un autre sera ouvert, ou si vous quittez l'application.

Ainsi le paramètre "Mettre en cache les fichiers ouverts" peut être positionné à "Aucun", "Fichiers audio and vidéo", ou "Tous".

Si vous sélectionnez "Aucun", les fichiers seront tous déchiffrés et passés à la volée aux autres applications.

Si vous sélectionnez "Tous", l'application déchiffrera tous les fichiers avant de les ouvrir.

Si vous sélectionnez "Fichiers audio and vidéo", seuls les fichiers audio et vidéo seront déchiffrés avant ouverture, car ce sont les types de fichiers qui posent généralement problème.

Le dernier paramètre permet simplement de choisir l'emplacement où seront déchiffrés les fichiers temporaires..

L'option "Cache interne de l'application" est la plus sûre, car, à moins que votre appareil ne soit rooté, aucune autre application ne pourra y accéder directement, mais l'espace de stockage y est généralement limité par le système, donc vous devrez peut-être choisir une autre option si vous devez lire de gros fichiers.

Récupérer les nouveaux fichiers distants

En haut de l'écran principal, à droite du nom de dossier actuel, se trouvent 2 boutons.

Le premier vous permet de vérifier s'il y a de nouveaux fichiers sur vos comptes distants.

S'il y en a, ils seront créés localement, et le download sera planifié. La couleur de ce bouton indique l'état de l'opération.

Le second bouton indique l'avancement de la synchronisation des fichiers (suppression, download et upload).

Appuyer sur ces boutons lancera la synchronisation si aucune n'était en cours (indicateur noir), ou affichera le dialogue de progression de l'action en cours (autres couleurs).

Notez que des icones indiquant ces états apparaitront aussi à côté de chaque fichier dans la liste.

Sur tous les dialogues de progression, vous trouverez généralement deux boutons, pour annuler l'action en cours, ou la mettre en pause, puis la reprendre plus tard.

Utiliser StorageCrypt depuis les autres applications

Si vous essayez d'ouvrir, sauver ou partager un fichier depuis une application pendant que l'application StorageCrypt fonctionne, les fichiers et dossiers chiffrés seront proposés comme sources ou destinations.

Si vous sauvez un fichier dans un de ces dossiers, il sera chiffré avec la clé par défaut du dossier en question.
