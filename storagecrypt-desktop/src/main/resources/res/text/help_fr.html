Création du magasin de clés

Lors du premier lancement de l'application, une clé de chiffement est générée et stockée dans un magasin de clés sécurisé.

Vous devez fournir un mot de passe, pour protéger ce magasin de clés. Ne l'oubliez pas, vous devrez le saisir à chaque fois que vous lancez l'application.

Associer vos comptes de stockage dans le cloud

Au début, vous n'avez qu'un seul emplacement pour chiffrer vos dossiers. Il est nommé "Fichiers non synchronisés". Comme vous pouvez le deviner, les fichiers chiffrés ici restent sur votre appareil.

Si vous souhaitez que vos fichiers chiffrés soient synchronisés avec Dropbox, par exemple, vous devez d'abord associer votre compte dropbox.

Pour cela, utilisez le bouton ("Associer un compte de stockage") dans la barre d'outils en haut de l'écran.

Vous pouvez choisir comme type de compte "Google Drive", "Dropbox", "Box", "HubiC" ou "One Drive".

Vous devez aussi sélectionner quelle clé de chiffrement sera utilisée par défaut pour ce compte (vous pouvez gérérer plusieurs clés ou en utiliser une seule, selon vos préférences).

Il vous est ensuite demandé de saisir les informations de connexion à votre compte.

Une fois votre compte associé à l'application, si des fichiers chiffrés s'y trouvent déjà, ils seront repérés et téléchargés.

Vous remarquerez alors un nouvel emplacement de stockage, aux côtés des "Fichiers non synchronisés", qui représente les fichiers synchronisés avec votre compte.

Chiffrer des fichiers

Pour chiffrer un fichier, vous devez auparavant vous placer à l'endroit où vous voulez le placer, par exemple "Fichiers non synchronisés".

Ensuite, utilisez le bouton ("Chiffrer") dans la barre d'outils en haut de l'écran.

Vous pouvez aussi faire glisser un ou plusieurs fichier depuis votre gestionnaire de fichiers.

Après la phase de sélection, vous devez choisir la clé avec laquelle le fichier sera chiffré. Vous pouvez utilisez différentes clés ou vous contenter de la clé par défaut pour tous vos fichiers.

Si la destination se trouve à l'intérieur du dossier "Fichiers non synchronisés", il ne se passera rien de plus.

Si elle se trouve à l'intérieur d'un emplacement associé à un compte de stockage dans le cloud, les fichiers qui y sont chiffrés seront synchronisés sur ce compte.

Download et upload des fichiers

Le transfert de fichiers peut impliquer une grande consommation de bande passante, il est donc possible d'attendre une connexion à un réseau wifi pour transférer les fichiers.

C'est la configuration par défaut de l'application, ainsi, quand un fichier est chiffré, il est marqué pour être transféré ultérieurement, lorsque vous serez connecté à un réseau wifi.

Notez que l'application doit être lancée (même en arrière plan) pour que les transferts se fassent.

Si voud quittez l'application, tous les transferts en cours seront abandonnés, et planifiés à nouveau pour le prochain lancement.

Créer des dossiers

Afin de gérer plus facilement vos fichiers chiffrés, vous pouvez créer des dossiers sur vos emplacements de stockage.

Pour cela, utilisez le bouton ("Créer un dossier vide") dans la barre d'outils en haut de l'écran.

Saisissez le nom du nouveau dossier, et sélectionnez la clé qui sera utilisée par défaut pour y chiffrer des fichiers.

Consulter les documents chiffrés

Vous pouvez naviguer dans les différents dossiers cliquant dessus. D'autres actions sont possibles avec un clic droit.

En haut de l'écran, est affiché le nom du dossier actuel, ainsi que la liste des dossiers parents.

Cliquez sur un des parents pour revenir à ce dossier.

Vous pouvez aussi cliquer sur le dossier actuel (tout à droite de la liste) pour afficher les actions supplémentaires.

Cliquez sur les fichiers chiffrés pour les ouvrir (ils seront déchiffrés automatiquement). Un clic droit sur les fichiers vous permettra d'exécuter d'autres actions.

Gérer vos clés

Si vous sélectionnez le menu "Gérer les clés", la liste des clés sera affichée.

Un clic droit sur l'une d'elle vous permettra de la renommer ou la supprimer. Renommer une clé ne vous empèchera pas de relire les fichiers que vous avez chiffrés avec.

Attention : si vous supprimez une clé, par contre, vous ne pourrez PAS récupérer les fichiers que vous avez chiffré avec.

Dans le menu, vous trouverez les options permettant d'exporter des clés, par exemple si vous voulez accéder à vos fichiers chiffrés sur un autre appareil.

Les clés que vous choisissez d'exporter seront sauvées dans un magasin de clés protégé par mot de passe.

Transférez ce magasin de clés sur votre autre appareil, et utilisez l'option "Importer des clés" pour importer les clés que vous souhaitez.

Dans ce menu, vous trouverez également une option pour générer une nouvelle clé, que vous pourrez utiliser pour chiffrer certains fichiers.

L'option "Changer le mot de passe" vous permet de changer le mot de passe du magasin de clés. Vous devrez saisir l'ancien mot de passe et le nouveau.

Paramètres de l'application

Si vous sélectionnez le menu "Paramètres", vous pourrez modifier les paramètres de l'application.

Sélectionnez le mode de connexion que vous souhaitez :

  • Pas de proxy
  • Utiliser les proxies du système
  • Configuration manuelle

Si vous choisissez la dernière option, vous devez également saisir l'adresse et le port du proxy

Récupérer les nouveaux fichiers distants

En haut de l'écran principal, à droite du nom de dossier actuel, se trouvent 2 boutons.

Le premier vous permet de vérifier s'il y a de nouveaux fichiers sur vos comptes distants.

S'il y en a, ils seront créés localement, et le download sera planifié. La couleur de ce bouton indique l'état de l'opération.

Le second bouton indique l'avancement de la synchronisation des fichiers (suppression, download et upload).

Cliquer sur ces boutons lancera la synchronisation si aucune n'était en cours (indicateur noir), ou affichera le dialogue de progression de l'action en cours (autres couleurs).

Notez que des icones indiquant ces états apparaitront aussi à côté de chaque fichier dans la liste.

Sur tous les dialogues de progression, vous trouverez généralement deux boutons, pour annuler l'action en cours, ou la mettre en pause, puis la reprendre plus tard.
